Инструкция по использованию Bypassing Prohibitions

Программа предназначена для автоматического подбора и запуска конфигураций обхода ограничений доступа к веб-ресурсам.
1. Основное окно (Активация):
   * Укажите ссылку: В верхнее поле вставьте URL-адрес ресурса, который недоступен (например, https://google.com).
   * Выбор режима:
   * Auto: Программа будет по очереди запускать все доступные .bat конфиги из папки core и проверять, появился ли доступ к сайту.
   * Ручной выбор: Выберите конкретный файл из выпадающего списка, если знаете, какой из них работает лучше.
   * Кнопка «Активировать»: Запускает процесс подбора. Все этапы (попытки запуска, проверки связи) отображаются в текстовом логе внизу.
   * Добавить в конфигурацию: Извлекает домен из вашей ссылки и сохраняет его в общий список list-general.txt, чтобы обход работал для этого сайта.

2. Настройки (Иконка ⚙):
   * Время подключения: Время в секундах, которое программа ждет ответа от сайта после запуска каждого конфига (по умолчанию 5 секунд).
   * Темы: Переключение между Темной и Светлой темами интерфейса.
   * Просмотреть конфигурацию: Открывает встроенный редактор списка доменов. Здесь можно вручную добавить, удалить или отредактировать адреса сайтов. Не забудьте нажать «Сохранить и выйти».

3. Особенности работы:
   * Права администратора: Программа автоматически запросит права администратора при запуске (необходимы для управления сетевыми службами).
   * Папка core: Все ваши рабочие скрипты должны находиться в папке core рядом с исполняемым файлом программы.
   * Автоматизация: При закрытии приложения или смене режима программа автоматически завершает работу фоновых процессов обхода (winws.exe), чтобы они не конфликтовали друг с другом.

------------------------------
Совет: Если автоматический подбор идет слишком быстро и не успевает сработать, увеличьте «Время подключения» в настройках.
